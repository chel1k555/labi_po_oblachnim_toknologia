## Работа

Сначала написал маленький код в файл main.cpp который буду использовать в image докерфайла.
```
#include <iostream>
#include <vector>

int main() {

    int a = 326;
    int b = 25266;
    std::vector<int> arr;

    arr.push_back(a * b);
    arr.push_back(a + b);
    arr.push_back(a - b);
    arr.push_back(a);

    for (int i = 0; i < arr.size(); ++i) {
        std::cout << "result - " << arr[i] << '\n';
    }
    
    return 0;
}
```
